**Voice Assistant (Python)**
This is a simple voice assistant made using Python.
It listens to your voice and gives responses.
**What it can do**
Say hello 
Tell its name 
Tell today’s date 
Stop when you say "stop" 
**Requirements**
Install these first:
pip install SpeechRecognition pyttsx3 pyaudio
**How to run**
python assistant.py
**Commands you can try**
"hello"
"your name"
"date"
"stop"
**Example**
You: hello  
Assistant: Hi Naga Lakshmi!
